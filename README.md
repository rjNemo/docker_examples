# 🐳 Docker examples

`Dockerfile` examples for:

- [Elixir](./elixir)
- [Go](./go)
- [Kotlin](./kotlin)
- [Node.js](./node)
- [Python](./python)
- [React](./react)

## 👷 Build a container

```sh
docker build -t $APPNAME:$VERSION .
```

## 🏃 Run a container

```sh
docker run --rm $APPNAME:$VERSION 
```
